{% if example.title == "Basic Cell Segmentation" %}
  • Uses the CellPose algorithm for accurate cell segmentation
  • Processes large images in chunks to optimize memory usage
  • Outputs results in a CSV format for easy downstream analysis
  • Can use pre-trained or custom models for different cell types
{% elif example.title == "Spatial Transcriptomics Alignment" %}
  • Aligns spatial coordinates between different modalities
  • Supports multiple transformation methods (rigid, similarity, affine, projective)
  • Works with standard pandas DataFrames for easy integration
  • Returns both the transformation matrix and quality metrics
{% elif example.title == "Multi-GPU Segmentation" %}
  • Distributes workload across multiple GPUs for faster processing
  • Uses Dask for distributed computing infrastructure
  • Stores intermediate results in Zarr format for efficient I/O
  • Optimized for very large whole-slide images (100k x 100k pixels and larger)
